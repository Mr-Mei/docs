## 前端常用布局方案

## 一、水平居中

**1. 使用text-align属性**

父元素设置 *text-align*: center，子元素设置 *display*: inline-block，实现水平居中

```html
<style>
    * {
        margin: 0;
        padding: 0;
    }
    :root {
        --c-size: 300px;
        --c-bgcolor: #ff8787;
    }
    .parent {
        height: var(--c-size);
        background-color: var(--c-bgcolor);
        /* 主要代码 */
        text-align: center;
    }
    .child {
        width: var(--c-size);
        height: var(--c-size);
        background-color: aqua;
        /* 主要代码 */
        display: inline-block;
    }
</style>
<body>
    <div class="parent">
        <div class="child"></div>
    </div>
</body>
</html>
```

**2. 定宽块级元素水平居中**

```html
<!--方法1 子元素设置 margin: 0 auto; -->
<style>
    * {
        margin: 0;
        padding: 0;
    }
    :root {
        --c-size: 300px;
        --c-bgcolor: #ff8787;
    }
    .parent {
        height: var(--c-size);
        background-color: var(--c-bgcolor);
    }
    .child {
        height: var(--c-size);
        background-color: aqua;
         /* 主要代码 */
        width: var(--c-size);
        margin: 0 auto;
    }
</style>
<div class="parent">
    <div class="child"></div>
</div>

<!--方法2 子元素设置 position: relative; 通过 left 属性 和 margin 实现 -->
<style>
    * {
        margin: 0;
        padding: 0;
    }
    :root {
        --c-size: 300px;
        --c-bgcolor: #ff8787;
    }
    .parent {
        height: var(--c-size);
        background-color: var(--c-bgcolor);
    }
    .child {
        height: var(--c-size);
        background-color: aqua;
        /* 主要代码 */
        width: var(--c-size);
        position: relative;
        left: 50%;
        /* margin-left 为 负的宽度的一半 */
        margin-left: -150px;
    }
</style>
<div class="parent">
    <div class="child"></div>
</div>
```
